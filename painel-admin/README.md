# Admin - NIMPI

Este é um projeto para controle de documentos da plataforma NIMPI usando Vue.js.

## Tecnologias utilizadas:
  - Node.js: v.10.15.0
  - NPM: v.6.4.1
  - Vue CLI: v.3.2.1
  - Vue.js: v.2.6.10
  - Vuex: v.3.0.1
  - Vue Router: v.3.0.3
  - Vue Resource: v.1.5.1


### Instalando as dependências e rodando o projeto:
```
npm install && npm run serve
```
